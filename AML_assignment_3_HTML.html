# Function to create sequences
def create_sequences(data, seq_length):
    X, y = [], []
    for i in range(len(data) - seq_length):
        X.append(data[i:i + seq_length])
        y.append(data[i + seq_length])
    return np.array(X), np.array(y)
# Normalize the temperature data

Three RNN-based models are implemented: * Model 1: Stacked LSTM with 64 and 32 units. * Model 2: Stacked GRU with 64 and 32 units. * Model 3: Conv1D (32 filters, kernel size 3) followed by an LSTM with 32 units.
